1. Create a program.


2. Create an updated version of that program.


3. New and old versions of the program.


4. Start the patch builder and enter the information about the patch.
Window title is the text that will appear in the title of the patcher window.
Game filename is the filename of the exe that will be patched. Checksum is an md5 checksum. The patcher will check the exe has the same checksum before patching. Leave this blank and the patcher will not check. Click the ... button to select the target exe and have the checksum generated for you.
Info license is the text that the patcher will display when run, use this to give information about your game and what the patch fixes.


5. Click the analyse button to show the analyse dialog.
Select the old and new exes.
The attached files will be shown as well as which files have changed.
Click Ok and the changed files will be added to the files listview in the main window.


6. Now press the build button and the patch will be built.


7. The completed patch running and waiting to patch.
